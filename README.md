# Поправка ✎

**Автоматическое согласование подлежащего и сказуемого в русском тексте.**

> **Статус: Бета** — проект рабочий, но на редких конструкциях возможны ошибки.

## Идея

Поправка работает как автозамена для русской грамматики: вы печатаете, а она следит, чтобы подлежащее и сказуемое были согласованы по роду и числу.

Написали «девушка *побежал*»? Поправка исправит на «девушка **побежала**». Написали «мы *пошёл*»? Поправка поправит на «мы **пошли**».

Редактор анализирует каждое предложение:

1. Разбивает на **клаузы** (части предложения, разделённые *и* или запятыми)
2. Находит **подлежащее** в каждой клаузе
3. Согласует с ним **глагол** по роду и числу
4. Подсвечивает исправленные слова

Понимает разницу между *«Маша и Петя пошли»* (составное подлежащее, мн.ч.) и *«Маша пошла, и Петя пошёл»* (две клаузы, разные подлежащие).

## Как это работает

- **Морфологический анализ** — библиотека [RussianNouns](https://github.com/georgy7/RussianNouns) + словарь спряжений глаголов
- **Разбиение на клаузы** — по союзу *и* или запятой, с проверкой глаголов с обеих сторон
- **Поиск подлежащего** — среди существительных и местоимений в именительном падеже до первого глагола в клаузе
- **Коррекция** — автоматическое изменение окончаний глаголов прошедшего и настоящего времени

## Возможности

- ✏️ **Автоисправление** — редактор сам находит и исправляет ошибки согласования
- 🎨 **Полный режим** — цветная разметка частей речи, клик по слову — выбор другой формы
- 📦 **Всё в браузере** — словари загружаются один раз, никакого сервера не нужно
- 🌐 **Работает офлайн** — после первой загрузки словарей можно отключить интернет

## Как использовать

Откройте `index.html` в браузере (или через локальный сервер) и начните печатать. Редактор будет подсвечивать исправления по мере ввода.

## Примеры

| Пишете | Получаете |
|---|---|
| Девушка купил чай | Девушка **купила** чай |
| Парень и девушка пошла | Парень и девушка **пошли** |
| Мы пошёл гулять | Мы **пошли** гулять |
| Оно открыла окно | Оно **открыло** окно |

## Технологии

- Чистый JavaScript (vanilla) — никаких фреймворков
- [RussianNouns.js](https://github.com/georgy7/RussianNouns) — для работы с существительными
- Собственный JSON-словарь спряжений русских глаголов
- Contenteditable — редактирование прямо в браузере
